Data Index Format (TSV)

The local search uses a TSV file at data/index.tsv. Columns:

typepackage | dataset | doc | tool | snippet
nameDisplay name
descriptionShort summary
tagsComma/semicolon/space separated tags
urlInternal URL (e.g., /docs/routing or /public/index.html)

Example

type	name	description	tags	url
package	SnackBox HTTP Server (C++)	Minimal raw TCP HTTP server with strict routing and static files.	c++;server;http;/public/index.html
doc	SnackBox Routing Spec	How strict routing and 404s are handled in SnackBox.	routing;spec;/docs/routing
